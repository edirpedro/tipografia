Parágrafos

Todos os parágrafos são envolvidos na tag p. Adicionalmente, os elementos p podem ser envolvidos por um elemento blockquote se o elemento p é de fato uma citação. Veja um exemplo de uso:

Nós somos o que fazemos repetidas vezes.

Aristóteles

Texto em linha

Existe uma variedade de elementos de linha no HTML que você pode usar no meio dos textos como negrito, itálico, taxado, código, entre outros. Veja uma lista de opções:

strong e b
Usado para enfatizar fortemente uma parte do texto colocando em negrito. A diferença entre usar strong e usar b esta no fato de que a primeira irá enfatizar o texto em leitores de tela para cegos e a segunda opção serve apenas como elemento decorativo, não carregando nenhuma importância no texto.
em e i
Usado para enfatizar uma parte do texto colocando em itálico. A diferênça entre usar em e i esta no fato de que a primeira irá enfatizar o texto em leitores de tela para cegos e a segunda opção serve apenas como elemento decorativo, não carregando nenhuma importância no texto.
cite
Usado para indicar uma citação ou referência para outra informação como o autor ou a fonte da informação.
code
Identifica uma linha de código em formatação de espaçamento único. Útil para inserir códigos de máquina que precisam ser perfeitamente lidos.
small
Define uma fonte de tamanho menor para o trecho do texto.
big
Define uma fonte de tamanho maior para o trecho do texto.
del
Usado para demarcar um trecho do texto removido mas que precisa ser mantido no texto. Exemplo: Ela comprou dois três pares de sapatos.
ins
Usado para demarcar um trecho inserido no meio do texto. Exemplo: Ela comprou dois três pares de sapatos.
q
Usado para citar um trecho do texto entre aspas. Exemplo: José disse a Maria: Você vem sempre aqui?
sup
Usado para demarcar um texto sobrescrito. Exemplo: 32 = 9.
sub
Usado para demarcar um texto subescrito. Exmplo: H2O.
abbr
Define uma abreviação de uma palavra maior ou sigla, como NASA.

Imagens

Ao inserir imagens no texto você pode escolher a sua posição especificando a classe alignleft para alinhar a imagem ao lado esquerdo do texto ou alignright para alinhar ao lado direito.

Listas

Existem três tipos de listas, a lista não ordenada, que utiliza marcação com símbolos como bolinhas, quadrados e outros, a lista ordenada que utiliza marcações como 1,2,3... a,b,c... i,i,iii... entre outras e a lista de definição composta por um termo e sua definição.

  • Elemento de lista não ordenada;
  • Que tem um segundo item;
  • E mais um terceiro.
  1. Elemento de lista ordenada;
  2. Que tem um segundo item;
  3. E mais um terceiro.
  1. Elemento de lista ordenada cascateada;
    • Onde este é o conteúdo relacionado não ordenado;
    • Que tem um segundo item;
      1. Que por sua vez relaciona outros itens ordenados;
      2. Que possui um segundo item;
    • E mais um terceiro.
  2. Que tem um segundo item relacionado à primeira lista;
  3. E mais um terceiro.

Tabelas

Tabelas devem ser usadas para mostrar dados tabulares apenas, nunca um layout. Os elementos thead, tfoot e tbody ajudam a agrupar corretamente as partes que compõem a tabela. O elemento caption define um sumário para introdução aos dados da tabela.

Ao utilizar tabelas nas páginas, você deve procurar usar todos os elementos de agrupamento respeitando sua ordem thead para o cabeçalho, tfoot para os dados de resultado e tbody para o conteúdo listado na tabela.

Bloco de código

Um bloco de código pode ser escrito usando o elemento pre para manter a correta tabulação da escrita pois aqui é usada uma fonte de espaçamento único, própria para texto de máquina.

function check(value) {
	if(value == 1)
		return 'success';
	else
		return 'error';
}
